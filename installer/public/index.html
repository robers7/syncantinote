Syncantinote Installer for macOS

This installer is designed to run directly from Downloads in Finder. It asks for your enrollment key, requests folder access as needed, installs Syncantinote.app, and configures startup sync on login.

Download Installer (.zip) Raw .command

If your browser warns about the file type, keep the file and run it from Terminal:

chmod +x ~/Downloads/SyncantinoteInstaller.command && ~/Downloads/SyncantinoteInstaller.command

If Finder says the file is not executable, unzip the ZIP first and run the extracted SyncantinoteInstaller.command. If your browser auto-unzips downloads, run the extracted file directly from Downloads.

The installer will:

  1. Prompt you for the enrollment key.
  2. Prompt for access to the Antinote Application Support folder.
  3. Download the latest helper from GitHub.
  4. Create Syncantinote.app in /Applications.
  5. Configure startup launch so sync runs automatically after login.
  6. Show a menu bar icon while running, with a single Close option to exit.
